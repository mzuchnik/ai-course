Kontekst Projektu

Projekt: Lexpage - aplikacja webowa zarządzania treścią prawną                                                                                                                                                                                                                                                     
Stack technologiczny:
- Spring Boot 4.0.1
- Java 25
- Gradle
- Thymeleaf (zależność już dodana, ale nieużywana)
- PostgreSQL + Liquibase
- Hexagonal Architecture z DDD

Obecny stan:
- Tylko REST API (@RestController), brak MVC controllers
- Katalogi src/main/resources/templates/ i src/main/resources/static/ są puste
- Brak jakiegokolwiek frontend toolingu (npm, webpack, etc.)
- Brak spring-boot-devtools
- Brak konfiguracji Thymeleaf w application.properties

Cele Integracji

1. Instalacja Material Tailwind HTML v2.3.2 przez npm z pełnym build process
2. Integracja Tailwind CSS v3.x z Material Tailwind preset
3. Material Icons z Google Fonts
4. Server-Side Rendering z Thymeleaf
5. Vanilla JavaScript (bez frameworków typu React/Angular)
6. Środowisko deweloperskie z hot-reload (Tailwind watch + Spring Boot DevTools + LiveReload)
7. Optymalizacje produkcyjne (Tailwind purge/minify + Thymeleaf caching)
8. Komponenty Thymeleaf (fragments) dla Material Tailwind z przekazywaniem parametrów
9. Mapowanie komponentów Material Tailwind:
- Buttons + Inputs + Forms
- Cards + Alerts + Badges
- Navbar + Footer
- Typography + Dividers
- Layout (Container, Grid, Section)

Wymagania Techniczne

- Node.js: v20.11.0 LTS (będzie pobrane automatycznie przez Gradle)
- npm: v10.2.4
- Tailwind CSS: ^3.4.1
- Material Tailwind HTML: ^2.3.2
- PostCSS: ^8.4.35
- Autoprefixer: ^10.4.17
- Gradle Plugin: com.github.node-gradle.node v7.1.0

const fs = require('fs');                                                                                                                                                                                                                                                                                          
const path = require('path');

// Copy Material Tailwind JS components to static/js                                                                                                                                                                                                                                                               
const sourcePath = path.join(                                                                                                                                                                                                                                                                                      
__dirname,                                                                                                                                                                                                                                                                                                       
'..',                                                                                                                                                                                                                                                                                                            
'node_modules',

);

__dirname,                                                                                                                                                                                                                                                                                                       
'..',                                                                                                                                                                                                                                                                                                            
'src',                                                                                                                                                                                                                                                                                                           
'main',                                                                                                                                                                                                                                                                                                          
'resources',                                                                                                                                                                                                                                                                                                     
'static',                                                                                                                                                                                                                                                                                                        
'js',                                                                                                                                                                                                                                                                                                            
'material-tailwind.js'                                                                                                                                                                                                                                                                                           
);

// Ensure directory exists                                                                                                                                                                                                                                                                                         
const destDir = path.dirname(destPath);

package pl.klastbit.lexpage.infrastructure.web.controller;

import org.springframework.stereotype.Controller;

/**

* Inbound adapter in hexagonal architecture.

  @Controller                                                                                                                                                                                                                                                                                                        
  public class PageController {

  @GetMapping("/")

  return "pages/index";

  return "pages/contact";                                                                                                                                                                                                                                                                                    
  }                                                                                                                                                                                                                                                                                                              
  }

Troubleshooting

Problem: Tailwind CSS się nie przebudowuje

Rozwiązanie:
- Sprawdź tailwind.config.js - czy content paths są poprawne
- Zrestartuj tailwindWatch task
- Sprawdź czy frontend/css/input.css istnieje

Problem: Komponenty Material Tailwind nie są stylowane

Rozwiązanie:
- Sprawdź tailwind.config.js - czy używa withMT() preset
- Sprawdź czy @material-tailwind/html jest w node_modules
- Uruchom npm run build manualnie do testów

Problem: Thymeleaf templates nie robią hot-reload

Rozwiązanie:
- Sprawdź czy spring-boot-devtools jest w dependencies
- Sprawdź application-dev.properties - czy cache=false
- Sprawdź czy używasz profilu dev: --spring.profiles.active=dev

Problem: Material Icons się nie wyświetlają

Rozwiązanie:
- Sprawdź layouts/base.html - czy link do Google Fonts jest obecny
- Sprawdź console przeglądarki (network errors)
- Sprawdź połączenie internetowe (CDN)

Problem: Ripple effect nie działa

Rozwiązanie:
- Sprawdź czy material-tailwind.js jest załadowany (Browser DevTools > Network)
- Sprawdź console przeglądarki (JavaScript errors)
- Sprawdź czy atrybut data-ripple-light jest obecny na buttonach

Problem: Gradle task fails

Rozwiązanie:                                                                                                                                                                                                                                                                                                       
./gradlew taskName --stacktrace
- Sprawdź zgodność wersji Node.js
- Wyczyść node_modules i reinstaluj: ./gradlew cleanFrontend npmInstall

Problem: npm install fails on Windows

Rozwiązanie:
- Uruchom jako administrator (permission issues)
- Sprawdź czy long path support jest włączony w Windows
- Użyj forward slashes w paths
